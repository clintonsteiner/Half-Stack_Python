Example Bundling of Package with NPM

Install Verdaccio

1 npm install verdaccio
2 verdaccio

Create example package

1 head -c 100M </dev/urandom > myfile
2
3 mkdir -p myPackage/files/
4 tar -czf ./myPackage/files/myPackage-1.0.0.tgz myfile

Create package.json file inside myPackage

{
"name": "myPackage",
"version": "1.0.0",
"description": "A generic package tarball",
"main": "index.js",
"files": [
  "files/myPackage-1.0.0.tgz"
]
}

Package and upload to local npm server

1 npm pack myPackage
2 npm publish myPackage --registry http://localhost:4873

Install and view package

1npm install myPackage --registry http://localhost:4873
2ls -grtha ./*myPackage*
